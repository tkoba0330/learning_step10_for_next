### 問題2: TypeScriptの型定義
以下のコードの空欄に入る適切な型定義を書いてください。

```typescript
type WeatherProps = {
  temperature: ______;  // 気温（数値）
  condition: ______;    // 天気（文字列）
  humidity: ______;     // 湿度（0-100の数値）
};
```

解答欄：
```typescript
type WeatherProps = {
  temperature: number;  // 気温（数値）
  condition: string;    // 天気（文字列）
  humidity: int;     // 湿度（0-100の数値）
};
```
